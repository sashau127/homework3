## списки

чтобы добавить ненумерованнные списки, необходимо пункты выделить звездочками, например вот так:
* элемент 1
* элемент 2
* элемент 3

чтобы добавить нумерованые списки, необходимо пункты просто пронумеровать. например вот так:

1. первый пункт
2. второй пункт
3. третий пункт
